## fluent API 
在 jquery 常见的链式调用方法，是通过返回对象的 this 实现的。
这种模式称为 [fluent API ](https://en.wikipedia.org/wiki/Fluent_interface) 你可以通过将 返回对象声明为 this 类型实现推导和扩展。
