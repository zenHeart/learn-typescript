# 对象类型-内建对象和字面量对象

本节讨论如何利用 TypeScript 限制内建对象和字面量对象。

## 对象分类
JavaScript 包含大量对象。大体分为三类

* **内建对象** 由 ECMAScript 规范定义
* **宿主对象** 取决于 JS 的执行环境,例如浏览器端的 Window,DOM。node 端的 Process,Global 等
* **自定义对象** 用户定义的对象

这里我们只讨论规范定义的内建对象和自定义对象,后续章节会逐步涉及在不同宿主环境下如何进行类型限制。
